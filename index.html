Midlands State University Smart Irrigation System Project

March - November 2024

Midlands State University, Gwer, Zimbabwe

This project focused on developing an Internet of Things (IoT)-based smart irrigation system tailored for Midlands State University (MSU). The system aimed to address water scarcity and inefficient water management in agriculture, particularly in dry and semi-arid regions like Gweru. By integrating advanced technologies such as soil moisture sensors, flow sensors, rainfall sensors, NodeMCU for microcontroller functionality, and Firebase for real-time data storage and analysis, the system provided real-time data for automated irrigation control.
